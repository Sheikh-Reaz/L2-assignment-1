This assignment project contains multiple TypeScript programming problems designed to strengthen understanding of:

- TypeScript fundamentals

- Interfaces and object typing
- Generics
- Union types and type guards
- Object-Oriented Programming (OOP)

- Clean and maintainable coding practices

All solutions are implemented inside a single file named:


# ⚙️ Installation & Setup

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/typescript-assignment.git
```

---

## 2️⃣ Navigate to the Project Folder

```bash
cd typescript-assignment
```

---

## 3️⃣ Install Dependencies

```bash
npm install
```

---

## 4️⃣ Run TypeScript Compiler

```bash
tsc
```

# ▶️ Run the Project

If you are using ts-node:

```bash
npx ts-node solutions.ts
```

Or compile manually:

```bash
tsc solutions.ts
node solutions.js
```
